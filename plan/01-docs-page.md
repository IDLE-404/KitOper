# 01 — Страница документации `/docs`

## Маршрут и файлы

- **URL:** `/docs`

## Содержание каждого раздела

### 1. Начало работы

**Подразделы:**

#### Что такое KitOper
Система управления расписанием колледжа. Основные роли:
- **Диспетчер** — полный доступ: составляет расписание, ведёт справочники, смотрит Форму 2
- **Преподаватель** — только просмотр своего расписания на сегодня
- **Студент** — базовый просмотр расписания

#### Как войти в систему
1. Перейдите по адресу системы
2. Введите логин (email) и пароль
3. Нажмите «Войти»

#### Боковое меню
Левая панель — навигация по разделам:
- **Расписание** — просмотр и редактирование
- **Дубликат недели** — копирование расписания
- **Генератор** — автоматическая генерация
- **Группы / Преподаватели / Дисциплины / Аудитории** — справочники
- **Праздники / Практика / Полевые лагеря / Отсутствия** — специальные периоды
- **Форма 2** — учёт фактических занятий
- **AI-ассистент** — чат с AI
- **Аудит** — журнал действий
- **Документация** — эта страница

### 2. Расписание — просмотр

#### Как открыть
Нажмите **«Расписание»** в боковом меню.

#### Выбор курса
Вверху страницы — выпадающий список курсов (1–4). Выберите нужный — страница перезагрузится.

#### Режим «Неделя» и режим «День»
- **Неделя** — все группы выбранного курса, пять дней одновременно, таблица
- **День** — только один день, удобно для большого числа групп или мобильного устройства

#### Числитель и знаменатель
Расписание чередуется каждую неделю:
- **Неделя A (числитель)** — первый вариант расписания
- **Неделя B (знаменатель)** — второй вариант расписания

В заголовке страницы написано, какой сейчас режим: «Сейчас показывается: неделя A (числитель)».

#### Навигация по неделям
Кнопки **«Предыдущая неделя»** / **«Следующая неделя»** или выбор даты в поле и кнопка **«Показать неделю»**.

#### Ячейка пары
Каждая ячейка показывает:
- Название предмета
- 👤 Преподаватель
- 🏫 Кабинет
- Для групп с подгруппами — строки 1 и 2

**Цвета ячеек:**
- Обычная белая — нормальная пара
- Жёлтая — замена (другой преподаватель или предмет)
- Красная — отсутствие (преподаватель болеет)
- Синяя пунктирная граница — пара выделена для навигации

#### Анализ недели
Кнопка **«Анализ недели»** проверяет конфликты: один кабинет у двух групп одновременно, отсутствующий преподаватель без замены. Красный значок — найдены проблемы.

#### Открыть/закрыть детали пары
Нажмите **✏️** в ячейке — откроется диалог с деталями и возможностью редактирования.

### 3. Расписание — редактирование

#### Два уровня редактирования
1. **Редактор недели** (`/schedule/week`) — создаёт шаблон без привязки к датам
2. **Развернуть семестр** — копирует шаблон на все даты семестра

#### Редактор недели
- Выберите группу
- Переключайтесь между числителем (A) и знаменателем (B)
- Нажмите на ячейку пары → откроется форма: предмет, преподаватель, кабинет
- Для группы с подгруппами — отдельные поля для подгруппы 1 и 2
- Нажмите **«Сохранить»**

#### Назначить замену
В форме редактирования пары нажмите **«+ Замена»**:
- Выберите заменяющего преподавателя
- При необходимости — замещающий предмет
- Укажите комментарий

#### Развернуть семестр
После настройки шаблона:
1. Прокрутите вниз к разделу **«Развернуть семестр»**
2. Укажите даты начала и конца семестра
3. Выберите курс и группы
4. Нажмите **«Развернуть»**

Система создаст записи для каждой реальной даты с учётом:
- Чередования числитель/знаменатель
- Праздничных дней (они пропускаются)
- Зимних каникул

#### Дублировать неделю
Раздел **«Дубликат недели»** в меню:
1. Выберите исходную неделю (откуда копировать)
2. Выберите целевую неделю (куда копировать)
3. Нажмите **«Дублировать»**

### 4. Форма 2

#### Что это
Официальный отчёт: сколько часов по каждому предмету фактически проведено за месяц. Данные поступают автоматически из расписания.

#### Как открыть
Нажмите **«Форма 2»** в боковом меню.

#### Выбор периода
1. Курс → Группа → Семестр → Месяц → Год
2. Нажмите **«ОК»**

#### Легенда ячеек
| Символ | Цвет | Значение |
|--------|------|----------|
| 2 | Фиолетовый | Пара проведена нормально |
| ■ | Жёлтый | Пара заменена другим предметом |
| 2 | Красный | Замена (бонусные часы) |
| • | Серый | Нет занятия |
| П | Жёлтый | Государственный праздник |
| (желт.фон) | — | День практики |
| Пунктир | Серый | Прогноз (Ghost-режим) |

#### Экспорт
- **📊 Экспорт в Excel** — текущий месяц
- **📘 Экспорт 1 семестр** / **📗 Экспорт 2 семестр** — весь семестр

#### Режим коррекции
Включите тумблер **«Режим коррекции»** для ручного редактирования:
- Изменить статус ячейки (нормально / замена / замещена)
- Добавить новую строку предмета (кнопка **«Добавить предмет»**)
- Изменить количество плановых часов
- После изменений нажмите **«Сохранить коррекцию»**

> **Важно:** Используйте коррекцию только при ошибках в расписании. Обычные данные приходят автоматически.

#### Просмотр семестра (Ghost 👻)
Включите переключатель **«Просмотр семестра»**. Система спрогнозирует часы до конца месяца на основе шаблона расписания. Прогнозные ячейки отображаются пунктирной рамкой — в базу не записываются.

### 5. Справочники

#### Преподаватели
- Список всех преподавателей
- Для каждого: ФИО, инициалы, кабинет по умолчанию
- **Важно:** у каждого преподавателя должны быть назначены предметы — только тогда он будет предлагаться при составлении расписания
- Оранжевая строка = дублирующая запись, нужно разобраться

#### Группы
- Список групп с указанием курса и типа (рус/каз)
- Флаг «подгруппы» — группа делится на 2 подгруппы
- **«Закончить год»** — перевести группу на следующий курс

#### Дисциплины
- Список предметов по курсам
- Для каждого: название на русском и казахском
- Редактирование сразу обновляет название везде

#### Аудитории
- Список кабинетов: код, тип (стандарт/компьютер/лаборатория), статус
- Неактивный кабинет не предлагается при назначении пар
- Используется функцией автоподстановки кабинетов

---

### 6. Специальные периоды

#### Праздники
- Добавьте дату → ячейки расписания заблокируются, Форма 2 пометит день жёлтым
- Праздник без года — повторяется ежегодно
- Праздник с годом — разовый

#### Практика
- Укажите группу, предмет, тип (учебная/производственная), даты
- В эти дни расписание показывает «На практике» / «Полевые сборы»
- Часов в день — учитывается в Форме 2

#### Отсутствия преподавателей
- Зафиксируйте болезнь, отпуск, командировку с датами
- В расписании ячейки с этим преподавателем помечаются красным
- При назначении пары система предупредит об отсутствии

### 7. Нагрузка преподавателей

Страница **«Занятость преподавателей»**:
- Выберите неделю
- Таблица покажет количество пар каждого преподавателя по дням
- Используйте для контроля перегруженности при составлении расписания

### 8. AI-ассистент

Встроенный чат работает на локальной AI-модели (Ollama):

**Что умеет:**
- Отвечать на вопросы о расписании на естественном языке
- Парсить Excel-файлы расписания и импортировать данные в систему

**Как использовать:**
1. Введите вопрос или запрос
2. Нажмите «Отправить»
3. Для импорта файла — нажмите кнопку загрузки, выберите `.xlsx`

**Примеры запросов:**
- «Какие пары у группы ИС-12 в среду?»
- «Импортируй расписание из этого файла»
- «Покажи нагрузку преподавателя Ахметова»

### 9. Аудит

Страница **«Аудит»** — журнал всех действий:
- Кто и когда вносил изменения
- Какой маршрут (раздел), метод, статус запроса
- Нажмите на строку — увидите полный payload (что именно изменилось)
- **«Откатить»** — вернуть состояние до этого действия (осторожно!)

### 10. Частые вопросы (FAQ)

**В: Данные в Форме 2 не обновляются после изменения расписания?**  
О: Данные синхронизируются при загрузке Формы 2. Откройте нужный месяц — данные обновятся. Если проблема сохраняется — проверьте аудит на предмет ошибок.

**В: В расписании показывается «неделя B», но я хочу посмотреть неделю A?**  
О: Используйте выбор даты — введите дату конкретной недели A и нажмите «Показать неделю». Система покажет именно эту неделю.

**В: Как добавить предмет в Форму 2 если его нет в таблице?**  
О: Включите «Режим коррекции» → нажмите «Добавить предмет» → выберите предмет и преподавателя → введите общее количество часов → «Сохранить коррекцию».

**В: При назначении пары преподаватель выделен красным?**  
О: Преподаватель числится как отсутствующий (болезнь, отпуск и т.д.) в этот день. Перейдите в «Отсутствия» и проверьте записи. Если ошибка — удалите лишнюю запись.

**В: Как работает Ghost-режим (Просмотр семестра)?**  
О: Ghost-режим показывает прогноз: если расписание продолжится по текущему шаблону, сколько часов будет отработано к концу месяца. Реальные данные остаются только до текущей даты. Прогноз в базу не пишется.

**В: Кнопка «Развернуть семестр» не реагирует?**  
О: Убедитесь, что шаблон недели заполнен хотя бы для одной группы. Проверьте, правильно ли указан диапазон дат (начало должно быть раньше конца).
