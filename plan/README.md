## Что будет сделано

### A. Страница документации `/docs`
Статичный Blade-шаблон с навигацией по разделам. Диспетчер найдёт ответ на любой вопрос без тура.

### B. Интерактивные туры (Driver.js)
На каждой странице — кнопка **«? Помощь»** в правом верхнем углу. При клике запускается пошаговый тур: Driver.js затемняет всё, вырезает нужный элемент и показывает подсказку. Пользователь жмёт «Далее» → следующий элемент.
