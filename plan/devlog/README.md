## Как вести дневник

Создавай файл `YYYY-MM-DD.md` после каждой сессии. Шаблон:

```markdown
# Сессия — DD месяца YYYY

## Что сделали
### Название фичи
Файл: `...`
Описание что именно поменяли и почему.

## Что проверено
- [ ] / [x] пункты

## Известные ограничения
- ...
```
